Sound Reference¶

Here you will find a full description of the sound API.

In order to simplify the playback of audio content, Xila offers a complete sound API. Indeed, the latter is capable of playing WAV, MP3, AAC files as well as audio streams. This API is based on the AudioI2S library by Schreibfaul.
