Software handle reference¶

Introduction¶

Here you will find a full description of the software handle.

In order to facilitate software management, Xila uses software handles. The latter contains the basic information of the software in question (Name, Icon) and pointers to a function that to boot the software. They are in a way the identity cards of the software.
