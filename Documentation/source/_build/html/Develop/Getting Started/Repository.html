Getting Started With The Repository¶

Here you will find instructions on how to setup the repository.

Requirements¶

  • A computer..

  • The latest version of Platform IO IDE.

  • The latest version of the Nextion editor.

  • The latest version of Git.

  • A GitHub account.

Instructions¶

Firstly, go on Xila github page.

Fork the project by clicking on fork.

After a short while, you will be redirected to the fork page. On this page, click on the Code button then copy the URL that is there.

Open a terminal and enter the following command :

git clone <url> <directory>

Then, open the directory where the repository was cloned with VSCode.

Now you are ready to contribute to Xila.
