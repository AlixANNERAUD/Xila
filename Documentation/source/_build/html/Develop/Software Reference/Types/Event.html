Event Type¶

Here you will find a full description the event type.

The various functions (for example: event dialogs, file management etc.) of Xila often return an Event type in order to indicate the state of an operation.

API Reference¶

enum Xila_Class::Event¶

Xila event type.

Values:

enumerator Success¶

Success event.

enumerator Error¶

Error event.

enumerator Warning¶

Warning event.

enumerator Information¶

Information event.

enumerator Question¶

Question event.

enumerator Button_1¶

Button 1 reply (equivalent to Yes reply).

enumerator Button_2¶

Button 2 reply (equivalent to No reply).

enumerator Button_3¶

Button 3 reply (equivalent to Cancel reply).

enumerator Default_Yes¶

Yes reply (equivalent to Button 1).

enumerator Default_No¶

No reply (equivalent to Button 2).

enumerator Default_Cancel¶

Cancel reply (equivalent to Button 3).

enumerator None¶

Example¶

if (Xila.Dialog.Event("Event dialog test", Xila.Information) == Xila.Default_Button_Yes)
{
    // -- Do stuff if the "Yes" button was pressed.
}

if (Xila.Account.Check_Credentials("Username", "Password") == Xila.Success)
{
    // -- Do stuff if the credentials are correct.
}
else
{
    // -- Do stuff if the credentials are invalid.
}
