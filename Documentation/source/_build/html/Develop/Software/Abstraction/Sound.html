Sound Abstraction Layer¶

Here you will find a full description of the sound abstraction layer.

This abstraction layer takes care of everything related to sound like the playback of audio (WAV, MP3, AAC, Stream etc.). This API is based on the AudioI2S library by Schreibfaul.

Warning

The AudioI2S currently does not have a stable version.
