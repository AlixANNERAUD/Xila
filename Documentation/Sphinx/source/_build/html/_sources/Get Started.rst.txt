Start To Use
============

What You Need
-------------

Hardware :

* An **ESP-32 DOIT DEVKIT V1** is recommanded, but all kind of ESP-32 will do the job.
* A USB USB Cable (USB A to USB B Micro).
* A computer
  
Software :

You have 

* Lastest version of Arduino IDE.
* The Arduino-Espressif 32 framework installed.


Hardware
--------

First of all, you will need to set up the hardware in order to install Xila.


Software
--------

First of all, if you have not installed Arudino IDE yet, `click here <https://www.arduino.cc/en/software>`_ to download the lastest version.

Then, to install the Arduino-Espressif 32 framework, open the Arduino IDE, go in ``File`` and ``Preferences`` and then paste in the ``Additional Boards Manager URLs`` field :

.. code-block::
    https://dl.espressif.com/dl/package_esp32_index.json
